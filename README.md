# Roundtable Presentation (Persian)

Built with [Marp CLI](https://github.com/marp-team/marp-cli).

## Site
- [میزگرد آنلاین](https://nightmachinery.github.io/roundtable_1/)

## Files

- `slides.md` — slide content in Persian
- `theme.css` — custom RTL + Persian font theme
- `index.html` — generated static slide deck (for GitHub Pages)
- `roundtable.pdf` — generated PDF version

## Build

```bash
npm install
npm run build
```

## Outputs

- HTML: `index.html`
- PDF: `roundtable.pdf`
